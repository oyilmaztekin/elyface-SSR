# Breadcrumb

for creating breadcrumb

### props
- **_`cat`_**: string
  - Category of the news
- **_`title`_**: string
  - title of the news
- **_`breads`_**: array
  - for custom breadcrumbs

### Usage

```javascript
import Breadcrumb from "@comp/breadcrumb/";
<Breadcrumb cat="gündem" title="iletişim" breads={["Mainpage","Rules", "Privacy"]} />
```
